 Inaccessibility in Public Transit Networks

This repository contains the code and data used in:

**"Inaccessibility in Public Transit Networks"**  
Katherine Betz (2025)  
https://arxiv.org/abs/2512.03766

## Overview

This project studies accessibility in the New York Metro using a network-based approach.

Accessible stations are represented as nodes in a graph, with edges defined by transit connectivity. The resulting network is analyzed to examine its structure, spatial distribution, and the role of central stations.

The repository is designed to support reproducibility of the results in the paper, including data processing, network construction, and computation of network statistics.

See `data/README.md` for details on datasets and sources.
